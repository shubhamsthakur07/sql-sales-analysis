SQL Sales & Revenue Analysis Project

Objective:
To analyze sales data and generate revenue and profitability insights using SQL.

Dataset:
Sample Superstore Dataset (Kaggle)

Key Analysis Performed:
- Total revenue and profit analysis
- Category-wise and region-wise revenue
- Top customers by sales
- High vs low value order segmentation
- Identification of loss-making orders

SQL Concepts Used:
SELECT, WHERE, GROUP BY, HAVING, ORDER BY, LIMIT,
CASE WHEN, aggregate functions, subqueries

Outcome:
The project demonstrates practical use of SQL for financial and business analysis.
